# 📊 Dashboard de Vendas em Excel

## 📌 Sobre o Projeto

Neste projeto desenvolvi um **dashboard de vendas no Excel** com o objetivo de transformar dados brutos em informações visuais claras e estratégicas.

A proposta do desafio é demonstrar como organizar dados, realizar cálculos e construir visualizações que permitam analisar o desempenho de vendas e apoiar a tomada de decisões baseada em dados.

O dashboard apresenta indicadores importantes como receita total, número de vendas, uso de cupons e análise de vendas por plano e método de pagamento.

---

## 🧰 Ferramentas Utilizadas

- Microsoft Excel
- Tabelas Dinâmicas
- Segmentação de Dados
- Gráficos Dinâmicos
- Fórmulas de agregação

## 📈 Indicadores do Dashboard

O dashboard apresenta os seguintes indicadores:

- 💰 Receita Total
- 🧾 Total de Vendas
- 📊 Receita Média
- 🎟 Total de Cupons Utilizados
- 📦 Vendas por Plano
- 💳 Vendas por Método de Pagamento
- 🔁 Clientes com Renovação Automática

---

## 📊 Visualizações Criadas

O painel apresenta gráficos que permitem analisar:

- Distribuição de vendas por plano
- Métodos de pagamento mais utilizados
- Percentual de renovação automática
- Receita total gerada

Além disso, o dashboard conta com **filtros interativos**, permitindo explorar os dados de forma dinâmica.
